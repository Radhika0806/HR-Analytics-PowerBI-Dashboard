### 1. HR Analytics Dashboard: Employee Attrition & Workforce Intelligence
An interactive Business Intelligence dashboard built using Power BI to analyze employee attrition, workforce demographics, compensation trends, tenure patterns, and job satisfaction metrics. Designed to support data-driven HR decision-making and strategic workforce planning.

### 2. Purpose

The HR Analytics Dashboard is a technical Power BI solution developed to measure attrition rate, analyze department-level turnover, evaluate salary impact on employee exits, and identify workforce trends using DAX-based KPIs and relational data modeling. The project demonstrates end-to-end data cleaning, transformation, modeling, and visualization skills required for Data Analyst roles.

### 4. Data Source

*Source*: Structured HR Employee Dataset (tabular workforce data for analytics purposes)

The dataset includes employee-level information such as:<br>
- Demographics (Age, Gender, Education, Marital Status)<br>
- Job details (Department, Job Role, Job Level, Years at Company)<br>
- Compensation (Monthly Income, Salary Bands)<br>
- Attrition Status (Yes/No)<br>
- Job Satisfaction & Performance Indicators<br>

The dataset is structured in a relational format to enable KPI calculation, filtering, and aggregation for business intelligence reporting.
